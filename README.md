# 🌌 About AssistNeo

AssistNeo is a futuristic AI-powered assistant built using Python and Flask.

It combines:

* AI chat
* Voice interaction
* Image generation
* Web search
* Modern cyberpunk UI
* Markdown rendering
* Syntax-highlighted code blocks

into one powerful assistant.

AssistNeo works on:

* 💻 Laptop
* 📱 Mobile
* 🌐 Browser

# ⚙️ Step-by-Step Installation

## 1️⃣ Install Python

Download Python:

https://www.python.org/downloads/

IMPORTANT:
During installation enable:

```text
✅ Add Python to PATH
```

---

## 2️⃣ Clone Repository

```bash
git clone https://github.com/shakthiyuvan01-create/AssistNeo.git
```

---

## 3️⃣ Open Project Folder

```bash
cd AssistNeo
```

---

## 4️⃣ Install Requirements

```bash
pip install -r requirements.txt
```

---

## 5️⃣ Create `.env` File

Create a file named:

```text
.env
```

Add:

```env
GEMINI_API_KEY=your_api_key

GITHUB_TOKEN=your_github_token

WEB_SEARCH_KEY=your_web_search_key
```

---

# ▶️ Run AssistNeo

```bash
python smith_web.py
```

---

# 📱 Open on Mobile

Connect:

* Laptop
* Mobile

to the SAME WiFi.

Open browser on phone:

```text
http://YOUR_IP_ADDRESS:5000
```

Example:

```text
http://192.168.1.3:5000
```
